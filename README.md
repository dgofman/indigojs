indigojs
========

Open terminal

1) Install NodeJS http://nodejs.org/download/

2) "npm install" (install node packages)

3) "npm start" (run node server)

4) Open browser copy and paste URL from console

5) Run "make account" / "make helloworld" / "make firststep"
