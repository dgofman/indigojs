Accordion:

The Accordion is a navigator made up of child containers declared as an array. Each child uses the parent container’s navigator button. This button, when triggered by the user, uses animation to expand the corresponding child container and collapse whichever one was currently being displayed. By default, only one child container is expanded at a time, however this can be altered such that multiple containers are simultaneously displayed by setting multiple = true.

Example: Default header

<%- code('accordion', {labels: `['Label1', 'Label2', 'Label3']`}) %> <%- $('accordion', {labels: ['Label1', 'Label2', 'Label3']}) %>

Example: Customizing content, multiple selection, and default view index

<%- code('accordion', {labels: `['Label1', 'Label2', 'Label3']`, contents: `['View1', 'View2', 'View3']`, multiple: `true`, selectedIndex: `1`}) %> <%- $('accordion', {labels: ['Label1', 'Label2', 'Label3'], contents: ['View1', 'View2', 'View3'], multiple: true, selectedIndex: 1}) %>

Example: Replacing content with available templates

<%- code('accordion', {labels: `['Label1', 'Label2', 'Label3']`, contents: `['View1', 'View2', 'View3']`, templates: `['#templ', null, '#templ']`}, templ) %> <%- $('accordion', {labels: ['Label1', 'Label2', 'Label3'], contents: ['View1', 'View2', 'View3'], templates: ['#templ', null, '#templ']}) %>

Button:

The Button component allows the user to easily interact with the interface and navigate through an application. By default the Button is rectangular, however the style attributes can be modified to change shape and color and to specify labels and icons to clearly convey what action will be carried out. DateField:

The DateField component provides the user with a convenient way to select dates. The DateField control is displayed as a text input field with a calendar icon on its right side. Upon clicking the icon, the user is presented with a CalendarControl, from which they can scroll through months and years to select a specific date. Once a date has been selected, the CalendarControl will collapse and the DateField control will be populated with the selected date. If the user does not select a date, they can click the icon to collapse the calendar control and leave the DateField in its previous state. The DateField component can be customized to include prompts and styling attributes.

    The ToggleBox control allows for displaying and hiding information from view. When the associated button is selected, the information in the ToggleBox will be displayed; when deselected it will be collapsed. It is possible to define multiple toggled content items to be displayed simultaneously by using the toggle-content class.

    Example: Defining Label and Toggle Content

    <%- code('togglebox', {label: `'Hello World 1'`, content: `'<p class="toggle-content">Lorem ipsum dolor sit amet...</p>'`}) %> <%- $('togglebox', {label: `${locales.common.helloworld} - 1`, content: `

    ${loremIpsum}

    `}) %>

    Example: Defining Template

    <%- code('togglebox', {label: `'Hello World 2'`, template: `'#templ'`}, templ) %> <%- $('togglebox', {label: `${locales.common.helloworld} - 2`, template: '#templ'}) %>

    Example: Defining multiple toggle contents

    Tooltip:

    The ToolTip functions to provide the user with helpful information. The ToolTip pops up when the user mouses over a particular element, providing information about that element. It can be customized with position settings and styling. If no porition is specified, the Tooltip will automatically position itself relative to its parent container's boundaries.

    Example: Setting tooltip top left

    <%- code('tooltip', {target: `'#topLeft'`, position: `'top-left'`}, `<%- $('text', {value: 'Top Left', id: 'topLeft'}) %>
